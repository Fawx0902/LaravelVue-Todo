# LaravelVue-Todo
## Installation Process
```
git clone --branch main https://github.com/Fawx0902/LaravelVue-Todo.git or simply download zip and unzip 
cd todo-laravel-vue

composer install
npm install

update .env (Database, Port, User, Pass, etc.)

php artisan migrate
php artisan db:seed

php artisan serve
npm run dev
```
